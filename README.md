# 划词翻译助手

一个优雅的浏览器划词翻译插件，支持单词详细释义和句子快速翻译，采用浮动卡片设计，不影响原网页浏览体验。

![版本](https://img.shields.io/badge/版本-2.0.0-blue)
![许可证](https://img.shields.io/badge/许可证-MIT-green)

## ✨ 功能特性

- **划词即译**：鼠标选中文字即可自动翻译
- **单词卡片**：单个单词显示详细释义（含词性标注）
- **句子翻译**：长句快速翻译，原文译文对照显示
- **优雅设计**：浮动卡片式界面，支持点击外部关闭
- **多引擎支持**：Google翻译、百度翻译、有道翻译、MyMemory
- **智能切换**：多引擎自动 failover，确保翻译可用
- **本地缓存**：翻译结果本地缓存，重复查询秒开
- **一键开关**：通过扩展图标快速启用/关闭翻译功能

## 🚀 快速开始

### 1. 启动翻译服务

```bash
cd translate-server

# 安装依赖
pip install flask flask-cors requests

# 启动服务
python server.py
```

服务将在 `http://localhost:5000` 启动。

### 2. 安装浏览器扩展

1. 打开 Edge/Chrome 浏览器，进入 `edge://extensions/` 或 `chrome://extensions/`
2. 开启「开发者模式」
3. 点击「加载解压缩的扩展程序」
4. 选择 `translate-extension` 文件夹

### 3. 使用翻译

- 在任意网页**选中英文单词或句子**
- 自动弹出翻译卡片
- 点击卡片外区域关闭

## 🔧 配置翻译引擎

默认使用 **Google翻译**（无需配置）。如需使用其他引擎：

### 百度翻译（推荐，学术词汇准确）

1. 访问 [百度翻译开放平台](https://fanyi-api.baidu.com/) 申请 API
2. 编辑 `translate-server/config.json`：

```json
{
  "baidu": {
    "appid": "你的APPID",
    "appkey": "你的密钥"
  }
}
```

3. 修改优先级（可选）：

```json
{
  "priority": ["baidu", "google", "mymemory"]
}
```

### 有道翻译

1. 访问 [有道智云](https://ai.youdao.com/) 申请 API
2. 编辑 `translate-server/config.json`：

```json
{
  "youdao": {
    "app_key": "你的应用ID",
    "app_secret": "你的密钥"
  }
}
```

## ⚙️ API 接口

### 翻译接口

```bash
POST /translate
Content-Type: application/json

{
  "text": "Hello World",
  "target_lang": "zh"
}
```

响应：

```json
{
  "success": true,
  "translation": "你好，世界",
  "source": "Hello World",
  "engine": "google",
  "engine_name": "Google翻译"
}
```

### 健康检查

```bash
GET /health
```

### 查看引擎状态

```bash
GET /engines
```

## 📝 注意事项

1. **服务必须先启动**：翻译功能依赖本地 Python 服务，使用前请确保 `server.py` 已运行
2. **Google翻译**：在中国大陆可能需要网络代理
3. **缓存机制**：翻译结果会缓存，重复查询不消耗 API 额度
4. **频率限制**：默认 60 次/分钟，超出会提示稍后再试
5. **文本长度**：单次翻译限制 1000 字符，超长会自动截断

## 🔒 隐私说明

- 所有翻译请求发送到本地服务（localhost），不会直接泄露到第三方
- 翻译内容仅用于获取结果，不会被存储或分析
- 百度/有道等第三方 API 的数据处理请查阅其隐私政策

## 🐛 故障排除

| 问题 | 解决方案 |
|------|----------|
| 提示"所有翻译引擎不可用" | 检查网络连接，或申请百度/有道 API |
| 提示"请求过于频繁" | 等待 1 分钟后重试 |
| 卡片不显示 | 检查服务是否启动，刷新扩展 |
| 翻译结果不准确 | 尝试申请百度翻译 API（学术词汇更准） |
