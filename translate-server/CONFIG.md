# 网页翻译服务 - 配置指南

## 支持的翻译平台

| 平台 | 特点 | 申请地址 | 免费额度 |
|------|------|----------|----------|
| **百度翻译** | 学术词汇准确，推荐 | https://fanyi-api.baidu.com/ | 标准版免费5万字符/月 |
| **有道翻译** | 通用翻译质量好 | https://ai.youdao.com/ | 新用户送50元体验金 |
| **DeepL** | 学术翻译最佳 | 网页免费版 | 免费但有限制 |
| **MyMemory** | 备用方案 | 无需申请 | 1000次/天 |

## 快速配置

### 方式一：申请百度翻译（推荐）

1. 访问 https://fanyi-api.baidu.com/
2. 注册/登录百度账号
3. 进入「管理控制台」→「开发者信息」
4. 复制 **APP ID** 和 **密钥**
5. 打开 `config.json`，填入：

```json
{
  "baidu": {
    "appid": "你的APPID",
    "appkey": "你的密钥"
  }
}
```

### 方式二：申请有道翻译

1. 访问 https://ai.youdao.com/
2. 注册账号，进入「控制台」
3. 创建应用，获取 **应用ID** 和 **密钥**
4. 打开 `config.json`，填入：

```json
{
  "youdao": {
    "app_key": "你的应用ID",
    "app_secret": "你的密钥"
  }
}
```

### 方式三：直接使用（不申请）

如果不申请任何API，会自动使用：
- **DeepL** 网页版（学术翻译质量好，但可能不稳定）
- **MyMemory** 备用（免费，但质量一般）

## 修改优先级

在 `config.json` 中可以调整翻译引擎优先级：

```json
{
  "priority": ["baidu", "youdao", "deepl", "fallback"]
}
```

系统会按此顺序尝试，第一个可用的返回结果。

## 启动服务

```bash
python server.py
```

启动后访问 http://localhost:5000/engines 查看各引擎状态。

## 故障排除

### 所有引擎都显示不可用

1. 检查是否正确填入API密钥
2. 检查网络连接
3. 查看控制台错误信息

### 翻译速度慢

- 这是正常的，免费API有速率限制
- 缓存会自动保存最近200条翻译

### 学术词汇翻译不准确

- **强烈推荐申请百度翻译API**，其学术词汇库较完善
- 或将DeepL优先级调高（适合学术论文）
